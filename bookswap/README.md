# BookSwap - Book Exchange Application

BookSwap is a Flutter application that enables users to exchange books with others in their community. The app provides a platform for book lovers to share their books and discover new reads through a swap system.

## Features

### Authentication & User Management
- User registration and login
- JWT-based authentication
- Profile management
  - View and edit profile information
  - Change password
  - Upload profile picture
  - Delete account
- Role-based access control (Admin and Regular users)

### Book Management
- Add new books to the platform
- Edit book details
- Delete books
- Search books by title, author, or category
- Filter books by various criteria
- View book details
- Upload book cover images

### Book Exchange System
- Create swap requests
- View received swap requests
- Accept or reject swap requests
- Track swap status
- View swap history
- review completed swaps

### User Dashboard
- View personal book collection
- Track active swap requests
- View swap history
- Manage profile settings
- View notifications

### Admin Features
- Admin dashboard
- User management
  - View all users
  - Add new users
  - Edit user details
  - Delete users
- Book management
  - View all books
  - Moderate book listings
  - Remove inappropriate content
- System monitoring
  - View active swaps
  - Monitor user activities
  - Generate reports

### Technical Features
- RESTful API integration
- Secure data storage
- Real-time updates
- Responsive design
- Cross-platform compatibility
